# Сайт-визитка

Это простой локальный сайт-визитка с красивыми анимациями и системой регистрации.

## Особенности

- Современный дизайн с анимациями
- Адаптивный интерфейс для всех устройств
- Система регистрации и авторизации
- Плавные переходы между секциями
- Анимированный фон с градиентами и частицами

## Структура проекта

```
├── index.html          # Главная страница
├── css/styles.css      # Стили сайта
├── js/main.js          # JavaScript функциональность
└── vercel.json         # Конфигурация для Vercel
```

## Развертывание на Vercel

Проект готов к развертыванию на Vercel. Просто подключите репозиторий к Vercel, и сайт будет автоматически опубликован.
